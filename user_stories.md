## Patient User Stories
- As a Patient, I want to register an account so that I can access the system securely.
- As a Patient, I want to log in to the system so that I can manage my appointments.
- As a Patient, I want to book an appointment with a doctor so that I can receive medical consultation.
- As a Patient, I want to view my upcoming appointments so that I can plan my visits.
- As a Patient, I want to cancel an appointment so that I can update my schedule if needed.

## Doctor User Stories
- As a Doctor, I want to log in to the system so that I can access my dashboard.
- As a Doctor, I want to view my daily appointments so that I can manage my schedule efficiently.
- As a Doctor, I want to update appointment statuses so that patients are informed correctly.
- As a Doctor, I want to view patient information so that I can provide proper treatment.

## Admin User Stories
- As an Admin, I want to manage users (patients and doctors) so that the system remains organized.
- As an Admin, I want to create, update, and delete doctor accounts so that the clinic staff is up to date.
- As an Admin, I want to view system-wide reports so that I can monitor clinic activity.
- As an Admin, I want to manage appointments so that the system operates smoothly.
